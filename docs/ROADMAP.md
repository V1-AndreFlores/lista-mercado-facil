# Roadmap

## MVP

- Criar estrutura Expo + TypeScript.
- Configurar navegação básica.
- Configurar Redux Toolkit.
- Configurar SQLite.
- Criar schema inicial.
- Executar seed do supermercado inicial editável.
- Criar cadastro/listagem de supermercados.
- Criar lista de compras.
- Adicionar itens à lista.
- Categorizar itens por regras locais.
- Agrupar e ordenar itens por setores do supermercado.
- Marcar itens como comprados.
- Salvar histórico local.
- Criar testes unitários do motor de categorização e ordenação.

## Pós-MVP

- Reutilizar listas anteriores.
- Sugerir itens frequentes.
- Aprendizado local por correção manual.
- Importação de lista por texto livre.
- Compartilhamento de lista.
- Sincronização opcional futura.
- Backend opcional para catálogo comunitário.
- Avaliação de IA generativa opcional.
